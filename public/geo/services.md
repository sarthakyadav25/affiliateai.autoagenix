## Generative Engine Optimization (GEO)

GEO is the practice of optimizing content for AI-generated search results. Unlike SEO, GEO focuses on becoming the cited source in LLM responses.

### GEO Techniques

| Technique | Implementation |
|-----------|----------------|
| Entity Markup | Schema.org JSON-LD |
| Semantic Structure | H1-H6 hierarchy, lists, tables |
| Fact Density | Data-rich, zero-fluff content |
| Citation Building | Academic-style references |
| llms.txt | AI crawler index file |

### GEO vs SEO Comparison

| Aspect | Traditional SEO | GEO |
|--------|-----------------|-----|
| Target | Google SERPs | AI-generated answers |
| Metric | Rankings, CTR | Citations, mentions |
| Content Style | Keyword-optimized | Fact-dense, structured |
| Technical Focus | Backlinks | Schema, semantic HTML |
| Outcome | Link clicks | Brand attribution |
